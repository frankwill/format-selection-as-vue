# Format Selection as VUE

Format the selected text within literal strings in Vue format. Very useful for Vue applications that do not use build stages or SFC's.

## Instruction

1. Press `Ctrl + Shift + P`
2. Type out the full name “FRANKWSB: Format Template String in Vue” or just the initials of the command name, then press `Enter`.

And that's it!🫡
